Moderation Commands Reference
!ban @user [reason] — Permanently ban a user
!kick @user [reason] — Kick a user from the server
!mute @user [minutes] — Timeout/mute a user
!unmute @user — Remove timeout from a user
!timeout @user [minutes] [reason] — Timeout a user
!warn @user [reason] — Give a warning
!warnings @user — Check a user's warnings
!clearwarnings @user — Clear all warnings
!purge [1-100] — Delete messages
!slowmode [seconds] — Set channel slowmode
!lock / !unlock — Lock or unlock a channel

AI Toggle
Enable AI Responses
When enabled, pinging the bot (@bot your question) will trigger a human-like AI response.
How AI Works

1. Make sure your Anthropic API key is set in ⚙️ Settings.

2. Enable AI above.

3. In any server, a member (or you) can ping the bot like:
@YourBot what is the weather today?
@YourBot write me a Python hello world script

4. The bot will respond naturally like a human.
